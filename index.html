A Simple Chess AI

Using chessboard.js and chess.js

Making Random Moves

Generating moveset using chess.js to handle the game data and then chessboard.js for visualisation


Using Minmax Depth 3 Algorithm

Current evaluation function uses only material value and sees no point in advancing pieces unless it results in material gain. This explains why sometimes it juggles the rooks behind the lines of pawns! Current time complexity is:

T(f) = b^d


Using Full Alpha-Beta Minmax Algorithm

The minMax tree is now being pruned using an alpha beta algorithm - this results in a best case performance of:

T(f) = sqrt(b^d)


Improved Positional Evaluation Function

Normalised the positional value matrix by dividing by largest absolute element's value so max absolute element value is 1. Evaluation of position is now:

posVal = materialVal[i][j] + 0.1 * positionalVal[i][j].

Positional value varies from piece to piece
